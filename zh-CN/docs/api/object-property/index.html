object.property

通过已注册的 PropertyAdapter，对一个显式白名单数值属性执行 Tween。

State: loading

参数​

参数类型默认值说明
propertystringalpha由 PropertyAdapter 注册并审核的属性名。
valuenumber1绝对目标值；relative 为 true 时表示增量。
durationnumber200Tween 持续时间，单位毫秒；为零时立即应用结束值。
easestringSine.easeOut通过 RuntimeAdapter 传给 Phaser 4 的缓动名称。
relativebooleanfalse将 value 视为相对捕获初值的增量。
restorestringnever恢复策略：never、onComplete、onStop 或 always。

结果与安全边界​

首个切片仅支持有限数值。任意点路径和原型访问会被拒绝。重叠反馈使用栈式所有权，完成或停止时的恢复不会覆盖较新的所有者。
